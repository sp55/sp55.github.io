初学HTML5

发表于 2016-12-12

关于H5

要学习HTML5,首先要知道三个东西 HTML、CSS、JavaScript。
如果把H5比作一个人，那么HTML就相当于身体，CSS就相当于衣服等装饰，而JavaScript则是这个人的行为。
所以学习H5一般都从HTML和CSS开始，就如同小孩先要学会走路才能做一些比较复杂的举动。

初识HTML

如上所示，这就是一个html的初始模样，通过观察我们可以看出html是以<元素></元素>的结构来构建的，可以看出，一个完整的html有两部分组成，head和body。
head一般link进一些css文件，上面的 便是这是一个utf-8类型的文件，utf-8是国际通用格式，按照这种格式，不会产生乱码现象，则是标题的头部，也就是这个网页的名字。
body就是文档的主体，文档的内容我们一般都写在这里面。

基本控件

就像其他语言一样，H5中有很多系统的控件。
其中最常用的就是div，类比来说，div相当于View，有稍微复杂一点的页面，我们一般都会分块，将一些Button、Label、TextField等控件摆放到View上，然后再调整View和上面的子控件的位置，而div就相当于这个View，成块的区域都可以放到div上。
如何区分不同的div呐？OC上有全局变量和加Tag两种方法，在H5中也有类似的方法，class和id，顾名思义，class表示这个元素的类名，id就相当于这个元素唯一的标识。
H5中有h1到h6六种标题标签，标题标签一般用于强调某个事情，新闻类居多，h1字体最大，h6最小。
p标签也很常用，Paragraph段落，顾名思义，这是一个段落标签，非常常用。
button按钮，和OC一样，可以设置文字和图片作为背景，可以添加点击事件。
a标签，这是一个很神奇的标签，这是一个超链接标签，可以实现Button类似的效果，关键词href。
img图片标签，和可以导入本地图片和网络图片，关键词scr。
ul/ol和li一般连载一块用，ul是定义无序列表，ol表示有序列表，而li则是无序或者有序列表中的单个项目，就像Cell或者Item。

基本语法

一段好的代码，总会有清晰的几部分，低耦合，易修改。所以一般情况下，我们在html中只填内容，在css中才修改样式。就像ViewController和View，Activity和XML的那样。

首先，我们要简单了解html

html有n多标签，根据实现的类型可以大体分为三大类
1.块级标签
也就是能独占一行的标签
能随时设置高度和宽度 比如 div p h1 h2 ul li等
2.行内标签(内联标签)
多个行内标签可以显示到一行，不能改变宽度和高度
宽度和高度取决于内容的尺寸 比如span a label
3.行内-块级标签
多个行内-块级标签可以显示在一行
能随时设置高度和宽度 比如input button

要想会写css，首先要知道几个事情。

1.可继承性
所有标签可继承 visiblility cursor
内联标签可继承 line-height font font-size font-family text-descoration
块级标签可继承 text-indent text-align

2.修改标签的显示类型
如上文提到的html的实现类型所说，我们知道了html有三种显示类型，如果想修改怎么办，css可以搞定这个。
css中有一个display属性，能修改标签的显示类型
none 隐藏标签
block 让标签变成块级标签
inline 让标签变成行内标签
inline-block 让标签变成行内-块级标签(内联-块级标签) 这个十分常用

3.如何居中
居中分为水平居中和竖直居中
(1)水平居中
行内标签和行内块-级标签 text-align: center
块级标签 margin: 0 auto （上下为0，左右自动）
(2)竖直居中
行内标签 行内-块级标签 块级标签
line-height == height
也就是设置line-height和height一致

4.css布局
默认情况下，所有的网页标签都在标准流布局中
从上到下 从左到右
脱离流布局的方法有
float属性
left 脱离标签流 浮动在父标签左边
rigt
position 属性和left right top bottom属性
fixed 生成绝对定位 相对于浏览器
static 默认值 没有定位 元素在正常的流中 忽略left bottom
absolute 生成绝对定位 相对于static定位以外的第一个父元素进行定位
relative 生成相当定位元素 相对于其正常元素定位

5.css优先级
(1)相同级别 就近原则&叠加原则
(2)优先级 id选择器 > 类选择器 > 标签选择器

选择器的针对性越强，它的优先级就越高。（后面加✅表示重要）
通配符选择符 * 0
标签 1 ✅
类 10 ✅
伪类 10
伪元素 10
id 100 ✅
important 1000 ✅
选择器的权值加在一起，大的优先，如果权值相同，后定义的优先
important > 内联 > id > 类 > 标签|伪类|属性选择 > 伪元素 > 通配符 > 继承

}
